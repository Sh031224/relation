# 천생연분
💕 커플들을 위한 커뮤니티

## 소개
Vue.js를 기반으로 제작된 커플들을 위한 커뮤니티 사이트 입니다.
커플들만의 익명 커뮤니티가 있어 서로서로 다양한 정보를 주고 받을 수 있습니다.

### 주요 기능

  - 💬 채팅 시스템
  - 📆 D-DAY
  - ✍ 커뮤니티 시스템
  - 🔧 반응형 웹
